## 🔍 컴퓨터의 구조
> * CPU: 중앙처리장치. 데이터를 처리하는 역할
> * Memory: 데이터를 저장하는 기능 수행
> * I/O: 컴퓨터와 외부 장치 사이에 데이터를 교환하는 기능 수행
> * System Bus: 위의 모든 컴퓨터 시스템 구성요소들을 연결하여 데이터 전송을 수행
